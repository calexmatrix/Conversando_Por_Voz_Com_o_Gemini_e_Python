# Conversando_Por_Voz_Com_o_Gemini_e_Python
Assistente de voz inteligente desenvolvido no Google Colab utilizando a API do Gemini. Integra captura de áudio em tempo real, processamento multimodal nativo (Speech-to-Text integrado ao modelo) e síntese de voz com gTTS para uma interação fluida em português.
